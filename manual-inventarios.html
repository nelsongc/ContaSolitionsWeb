🏢 Introducción al Sistema

Sistema de Inventarios SISMER v.5

SISMER es el módulo de control de inventarios de la plataforma SISGES®, diseñado para ejercer un control integral del proceso de comercialización: ingresos por producción/compras y egresos por ventas/remisiones.

Figura 1: Pantalla principal del Sistema de Inventarios v.5 SISMER mostrando las opciones de ARCHIVOS PRINCIPALES, MOVIMIENTO INVENTARIO y MÓDULO COBRANZAS

🎯 Características Principales

  • Sistema de Costeo FIFO: El primero en entrar es el primero en salir
  • Libro de Compras IVA: Activación automática con numeración de facturas
  • Gestión de Clientes: Catastro completo con zonificación y sectorización
  • Control de Proveedores: Integración con plan de cuentas contable
  • Módulo de Cobranzas: Control de ventas al crédito y pagos
  • Reportes Integrales: Kardex, existencias, estados de cuenta

🔄 Integración con SISGES®

Todas las transacciones de inventario se integran automáticamente con la contabilidad, actualizando cuentas de inventarios, proveedores, clientes y resultados.

🧭 Navegación del Sistema

⌨️ Teclas de Función Universal

[Enter]

Avanza entre campos de izquierda a derecha y de arriba hacia abajo

[F1] - Ayuda

Búsqueda de códigos por denominación o descripción

[F2] - Retroceso

Navegación inversa entre campos para correcciones

[Esc] - Salida

Abandono del programa actual y retorno al menú

⚠️ Importante

Hasta que el sistema no requiera confirmación de grabación en la línea de mensajes, todas las acciones anteriores no alteran los archivos.

👥 Gestión de Clientes

🎯 Objetivo

Habilitar, modificar y/o eliminar el catastro de clientes. Esta registración es obligatoria si el sistema es habilitado con la gestión de cobranza.

🔢 Código

Campo numérico generado automáticamente para identificar persona natural o jurídica

👤 Nombre o Razón Social

Campo alfanumérico de 40 posiciones. Formato: Paterno, Materno y Nombres

📍 Lugar

Campo numérico de 2 posiciones para zonificación de clientes

🏭 Sector

Campo numérico de 2 posiciones para sectorización productiva

📄 N.I.T.

Número Interno de Tributación (12 posiciones) para Libros IVA

📞 Contacto

Información de contacto: teléfono, celular, email, dirección

Figura 2: Pantalla de captura de clientes con campos de código, nombre, lugar, sector, N.I.T. y datos de contacto
Figura 3: Sistema de búsqueda universal para localizar clientes por criterios de búsqueda alfanuméricos

🔍 Sistema de Búsqueda Universal

El sistema incluye una herramienta de búsqueda exclusiva que permite ubicar códigos por criterios no mayores a 10 caracteres alfanuméricos, recuperando todas las denominaciones que coinciden parcial o totalmente.

📥 Ingreso de Proveedores

🎯 Objetivo

Habilitar, modificar y/o eliminar el ingreso de mercaderías al sistema por concepto de compras, importaciones u otros que contienen precio de costo.

💰 Sistema de Costeo FIFO

El sistema adopta el método FIFO (PEPS): el primero en entrar es el primero en salir, sea por fecha de compra o vencimiento del producto.

📄 Parte Documentaria

  • Fecha de transacción
  • Concepto y referencias
  • Documentos de respaldo
  • Valores totales y descuentos

💼 Parte Transaccional

🏪 Campos de Compra

🏢 Empresa/Lugar

Código de almacén (11-99), números inferiores reservados para producción

📅 Fecha

Campo no digitable, recupera fecha del sistema automáticamente

🏭 Proveedor

Código contable de 8 posiciones del plan de cuentas SISCON

📄 No. Factura

Documento de referencia alfanumérico de 10 posiciones

⏰ Vencimiento

Fecha para integración con módulo de cobranza de acreedores

💱 T/C

Tipo de cambio. Valor 1 para facturas en Bolivianos

📦 Creación de Productos

🆕 Nueva Mercadería [F7]

El sistema permite crear nuevos códigos de productos durante el proceso de compra, con numeración automática y ascendente de hasta un millón de posibilidades.

🔢 Código

6 posiciones numéricas, generación automática por el sistema

📝 Descripción

Denominación del producto, campo alfanumérico de 35 posiciones

📏 U/M

Unidad de Medida abreviada de 3 posiciones (Pza, Kg, Lt, etc.)

📊 Línea

Clasificación genérica de productos por categoría (2 dígitos)

🏠 Propiedad

1=Mercadería Propia, 2=En Consignación

💰 Moneda

1=Bolivianos, 2=Dólares (recomendado crear todo en Bs.)

Figura 5: Pantalla de creación de nuevos productos con código, descripción, U/M, línea, propiedad y moneda

💲 Gestión de Precios

💰 Precio de Costo

6 posiciones enteras y 2 decimales. Opción (P)recio o (S)ubtotal

🏷️ Precio de Venta

Calculado automáticamente con porcentaje de ganancia parametrizado

📦 Precio por Mayor

Precio alternativo con porcentaje diferenciado

📦 Precio por Caja

Precio para ventas al por mayor

🔥 Precio de Liquidación

Precio especial para liquidación de inventarios

⚠️ Control de Totales

Los totales de control deben corresponder al valor total del documento. Este valor se utiliza en el libro de compras y control de cobranzas.

📤 Notas de Remisión

🎯 Objetivo

Habilitar, modificar y/o eliminar las salidas de productos por cualquier concepto menos ventas facturadas.

  • Transferencias entre almacenes
  • Muestras y degustaciones
  • Devoluciones y garantías
  • Mermas y ajustes de inventario
  • Préstamos temporales
Figura 6: Pantalla de nota de remisión para salidas de productos sin facturación, con cliente, fecha y detalle de productos

🔄 Integración Automática

Todas las remisiones actualizan automáticamente los saldos de inventario y se reflejan en los reportes de existencias y kardex de productos.

Todos los reportes incluyen filtros por fecha, cliente, producto, y almacén. Generación automática con opciones de exportación y formatos personalizables.
